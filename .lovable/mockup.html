🔌 Platform Integrations Each business connects its own accounts — manage MarzPay and SMS under Settings
ℹ️MarzPay (payments) and SMS gateway are configured per-business under Settings → Payments and Settings → SMS. The integrations below (Domain, Email, CHR) are also per-business and live here in Admin since they're less frequently touched.
📧 Email GatewayNot Connected
ℹ️Powers real email delivery for withdraw-passcode verification codes and any other account emails (currently those codes only show in an on-screen notification — connect this to actually send them).
Must be a verified sender on your provider
API Endpoint Used (SendGrid)
POST https://api.sendgrid.com/v3/mail/send
SendGrid requires Sender Authentication (verify your From Email) before it will deliver — see provider docs.
🌍 Domain Provider — IONOSNot Connected
ℹ️IONOS' API manages DNS records on domains you already own there. Use this to automatically point your hotspot/captive-portal or CHR hostname (e.g. hotspot.yourdomain.com) at your server's IP — it does not register new domains via chat.
API Endpoint Used
POST https://api.hosting.ionos.com/dns/v1/zones/{zoneId}/records
Generate an API key at IONOS Cloud Panel → API Keys. Keys are scoped per-contract.
📡 CHR MikroTik Setup WizardNot Connected
ℹ️CHR (Cloud Hosted Router) is MikroTik's RouterOS image for cloud VMs (AWS, DigitalOcean, IONOS Cloud, etc). This wizard walks through deploying and connecting one as your central API/RADIUS endpoint.
🖥️ Remote Access — WinBox0 routers
ℹ️Browsers can't open a direct WinBox session — clicking "Open in WinBox" launches the WinBox app already installed on this computer (via the winbox:// link) and pre-fills the address. If WinBox isn't installed, use the download link first.
